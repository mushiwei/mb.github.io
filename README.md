# Vue3 + Arco Design CRUD 应用

这是一个基于Vue3和Arco Design的日志管理CRUD应用。

## 功能特性

- ✅ 完整的CRUD操作（创建、读取、更新、删除）
- 🔍 搜索功能
- 📊 数据表格展示
- 🎨 响应式设计
- 📅 日期选择器
- 🏷️ 状态标签（成功/失败）

## 技术栈

- Vue 3.4.27
- Arco Design Vue 2.53.0
- 纯前端实现，无需后端

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. 本地运行
```bash
# 使用Python HTTP服务器
python -m http.server 8080

# 或使用Node.js的http-server
npx http-server
```

3. 访问应用
打开浏览器访问 `http://localhost:8080`

## 部署

本应用已配置GitHub Pages，可以直接访问线上版本。

## 功能说明

- **新增记录**: 点击"新增记录"按钮，填写表单信息
- **编辑记录**: 点击记录行的"编辑"按钮
- **删除记录**: 点击"删除"按钮并确认
- **搜索功能**: 在搜索框中输入名称进行筛选
- **重置功能**: 点击"重置"按钮清除搜索和分页
